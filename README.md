# Reservation Status Classification

This project is focused on *Reservation Status Classification*, where the goal is to predict the reservation status of bookings into three classes:  
- *Canceled*  
- *Checked-out*  
- *No-show*  

---

## Project Structure

- *data/* – Contains the datasets used in this project.  
  - Train/ – Train dataset.  
  - Validation/ – Validation dataset.  
  - Test/ – Test dataset.  

- *src/* – Contains the notebooks and scripts for data preprocessing and model training.  

---

## Usage

1. Explore, preprocess and Train the data using the notebooks in the src/ directory.
