# Hotel Chain A – Booking Cancellation Prediction
**Module:** 5DATA004C Data Science Project Lifecycle (2025/26)
**University:** University of Westminster
**Group:** B10
## Project Overview
Hotel Chain A operates City Hotels, Airport Hotels, and Resorts and has
been experiencing revenue losses due to booking cancellations and no-shows.
This project analyses historical booking data to uncover cancellation
patterns and builds a predictive model to classify reservation outcomes:
Check-Out, Cancelled, or No-Show.
## Repository Structure
- - - `data/` – Training and validation datasets
`coding files/eda_files_train_groupb10.py` – Exploratory Data Analysis
`coding files/modeltraining_groupb10.py` – Model training (Decision Tree + Neural
Networks)
- `requirements.txt` – Python dependencies
## Models Built
- Decision Tree Classifier
- Neural Network with Weighted Cross Entropy (Model I)
- Neural Network with SMOTE oversampling (Model II)
- Hybrid: SMOTE + Class Weights (Model III)- Decision Tree + SMOTE
## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run EDA: python coding files/eda_files_train_groupb10.py
3. Run model training: python coding files/modeltraining_groupb10.py
